What is Lumo?

Lumo is a simple programming language created for beginners. Its goal is to make programming easy to read, easy to write, and fun to learn.

Commands

Command Description
print Print text to the output.
var Create a variable.
if Start an if statement.
else Else block.
end End a block.
repeat Repeat a block multiple times.
break Exit a loop.
continue Continue to the next loop iteration.
clear Clear the output.
wait Pause execution.
message Display a message.
open Open a file.
write Write a file.
append Append text to a file.
delete Delete a file.
copy Copy a file.
